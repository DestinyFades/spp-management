# 🚀 SPP Management - Система управления СПП

## 📋 Описание
Микросервисное веб-приложение для управления иерархическим справочником элементов СПП с историчностью изменений, привязкой отделов и функционалом каскадного распределения сумм по структуре с поддержкой обновления и экспорта.

## 🏗️ Архитектура
- **Frontend**: Next.js 14 (React) + TypeScript
- **Backend**: FastAPI (Python 3.11)
- **База данных**: PostgreSQL 15 (3NF, SCD Type 2)
- **Кеш**: Redis 7
- **Real-time**: Server-Sent Events (SSE)
- **Контейнеризация**: Docker Compose

## 🔧 Схема работы Redis → БД → Real-time
1. **Распределение** → результат сохраняется в Redis с TTL (1 час)
2. **Сохранение** → перенос из Redis в PostgreSQL (JSONB)
3. **Real-time** → SSE уведомления всем клиентам сессии

## 🧪 Тестирование

1. Откройте http://localhost:8000/docs
2. Протестируйте эндпоинты

### Тестовые сценарии
1. **Выбор даты** → дерево перестраивается
2. **Выбор элементов** → чекбоксы работают
3. **Распределение суммы** → появляются суммы у узлов

## 🐛 Известные проблемы и решения

### Ошибка: Ошибка подключения к PostgreSQL
**Решение**: Убедитесь, что PostgreSQL полностью запущен (healthcheck: healthy)

### Ошибка: Ошибка BOM в JSON файлах
**Решение**: Создавайте файлы с кодировкой UTF-8 без BOM

### Ошибка: Медленная загрузка образов Docker
**Решение**: Используйте зеркала в Docker Desktop
